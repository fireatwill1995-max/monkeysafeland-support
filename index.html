Monkey Safe Land — Support

Help and troubleshooting for the Meta Quest build

Getting Started

Monkey Safe Land is a VR action game for Meta Quest. The current build supports solo play and direct host/join multiplayer over your local network.

Installation

  1. Enable Developer Mode on your Quest headset (Settings → System → Developer).
  2. Connect your headset to your PC via USB-C.
  3. Accept the USB debugging prompt on the headset.
  4. Install using SideQuest, or run:
    adb install -r MonkeySafeLand_Quest_TestBuild.apk
  5. Launch the app from your Quest library under Unknown Sources.

Controls

  • Left Stick — Move
  • Right Stick — Snap turn
  • Grip — Sprint
  • A — Start / Ready / Jump
  • X — Host a multiplayer session
  • B — Join / Disconnect
  • Y — Edit join IP address

Solo Play

  1. Launch the app.
  2. Press A to start a solo run.
  3. Collect objective pickups while avoiding the Shriek hunter.
  4. Complete the round before time runs out or your health reaches zero.

Solo play does not require a network connection.

Multiplayer Setup

The current build uses direct host/join networking. Both headsets must be on the same local network.

Hosting

  1. Press X to host. Your IP and port will appear on screen.
  2. Share the displayed IP with the joining player.
  3. Wait for the other player to join and ready up.
  4. Press A to launch the match once all players are ready.

Joining

  1. Press Y to open the join-IP editor.
  2. Enter the host's IP address. The default port is 27015.
  3. Press B to connect.
  4. Press A to ready up.

Troubleshooting

The app does not appear on my Quest

After sideloading, the app appears under Unknown Sources in your Quest library, not in the main app list. Make sure Developer Mode is enabled.

adb install fails or the headset is not detected
  • Reconnect the USB-C cable and look for the USB debugging prompt on the headset.
  • Try a different USB port or cable.
  • If using wireless adb, pair over USB first, then switch to tcpip mode with adb tcpip 5555.
Cannot join a multiplayer session
  • Confirm both headsets are on the same Wi-Fi network.
  • Double-check the host IP shown on the host headset.
  • Make sure no firewall is blocking port 27015 (or the port shown by the host).
  • If using different subnets (e.g., guest vs. main Wi-Fi), move both devices to the same network.
Performance feels low or the app stutters
  • Close other running apps on the headset.
  • Try lowering the in-game graphics quality or target FPS in the settings menu.
  • Restart the headset if it has been running for a long session.
My profile or progress was lost

Profile data is stored locally on the headset. Uninstalling or clearing the app's data will remove your saved profile. There is no cloud backup in this build.

Game Modes

  • Tag — Fast-paced chase mode. Evade the Shriek and survive the round.
  • Hide & Seek — Use map cover to stay hidden from the hunter.
  • Battle Royale — Compete under increasing zone pressure until one player remains.
  • Horror — Darker atmosphere, denser fog, and a more aggressive Shriek.

Available Maps

  • Jungle Canopy
  • Jungle Ruins
  • Volcanic Island
  • Abandoned Facility
  • Snowbound Village

Known Limitations (Current Build)

  • Multiplayer is direct host/join only — no platform matchmaking or relay servers.
  • Voice chat and free text chat are not included.
  • Profile data is local only with no cloud sync.
  • The package name com.yourstudio.monkeysafeland is a development placeholder.
